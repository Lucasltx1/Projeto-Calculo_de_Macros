# Macro Tracker

Sistema de controle nutricional e rastreamento de macronutrientes.

🔗 **[Acesse o sistema online](https://dieta.mandetalucas.workers.dev)**

## Sobre
O Macro Tracker é uma aplicação full-stack desenvolvida para facilitar o registro de consumo diário de alimentos e o acompanhamento de macronutrientes (proteínas, carboidratos e gorduras).

## Tecnologias
* **Backend:** Python, FastAPI
* **Frontend:** HTML5, CSS3, JavaScript
* **Banco de Dados:** Supabase (PostgreSQL)
* **Deploy:** Render (Backend) e Cloudflare Pages (Frontend)
